Regular Expressions

e
The character (or letter) "e".
ussare
The character sequence (string) "ussare".
*
Matches the previous character or group zero or more times.
+
Matches the previous character or group one or more times.
?
Matches the previous character or group zero or one time.
|
Use between characters or groups to mean a or b: a|b.
.
Matches any character.
\
Escape for special characters. you will have to escap the backslash with another one, \\ to represent a single backslash, and in Java and other programming languages, to match "c:\files\temp" you will have to write c:\\\\files\\\\temp. Four backslashes to match a single one.

Character classes

Pre-defined character classes

A few character classes have "short cuts" to simplify usage:

\d
A single digit (0 - 9).
\D
A single non digit (anything but 0 - 9).
\s
A single white space character (space, carriage return/new line, tab or form feed).
\S
A single non white space character.
\w
Any alphanumeric character including underscore.
\W
Any non alphanumeric character including underscore.

Defining a character class

Defining your own character classes.

[]
Used to create a character class: [a-cA-C] letters a, b and c in both lower and upper case.
[ABC123]
Upper case letters A - B or digits 1 - 3.
[\dA-F]
Upper case letters A - F or any digit (hexadecimal).
[^qwerty]
Not the lower case letters q, w, e, r, t or y.
gr[ae]y
The word grey spelled with a or e

Specify number of characters or groups (quantifiers)

{n}
Exactly n number of occurrances: A{3} matches "AAA", [123]{2} matches any combination of two of the numbers 1 2 and 3: "11", "12", "13", "21" etc.
{n,m}
Matches n through m number of occurrences: A{2,3} matches "AA" and "AAA", [123]{1,3} matches any combination of one through three of the numbers 1 2 and 3: "1", "12", "313", "22" etc.
{n,}
N or more: L{3,} matches "LLL", "LLLL", "LLLLLLLL" etc.

Groups

(x)
Match "x". This is a capturing group, so you can get the data from this one to process further: ([ABC]{3}) ([123]{}) matches a string with three upper case letters A-C, a space and three digits 1-3: "CCA 113", "ACB 221" etc. The first group will contain the letters and the second the digits.
(?:x)
Match "x". This is a non capturing group.
